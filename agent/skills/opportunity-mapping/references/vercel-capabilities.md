# Vercel capabilities (reference)

Use these as the capability vocabulary for opportunity mapping.

- First-class Next.js support, App Router, React Server Components
- Instant previews for every PR
- Global edge network for static and dynamic workloads
- Image optimization, fonts, and performance defaults

## AI Cloud

- AI SDK for multi-provider model apps
- v0 / AI-assisted UI generation workflows
- Streaming, tool calling, and agent-friendly primitives
- Fast iteration for AI product surfaces

## Fluid compute & infrastructure

- Serverless / fluid compute without managing servers
- Fast cold starts and automatic scaling
- Cron, workflows, and background work patterns
- Secure environment and deployment isolation

## Developer experience

- Git-connected deployments
- Preview URLs for design/product review
- Observability and logs for production debugging
- Team collaboration on deploys and comments

## Enterprise platform

- SSO, audit logs, and access controls
- Security and compliance posture for regulated buyers
- Support for multi-team platform standardization
- Migration and architecture guidance
