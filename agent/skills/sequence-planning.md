# Sequence planning

Design the outreach plan, not the copy.

Include:

- `channels` — ordered channel list (email, LinkedIn, etc.)
- `cadence` — overall rhythm
- `timing` — when to start / preferred windows
- `steps` — ordered steps with `channel`, `delayDays`, `purpose`
- `followUpLogic` — how replies and silence are handled
- `exitConditions` — when to stop, hand off, or nurture

For v1, plan as if only the first email will be drafted now. Later steps can remain planned.

Persist with `save_stage_output` stage `sequence_planning`.
