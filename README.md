# Lead Agent

A simple Next.js App Router dashboard for reviewing and prioritizing sales leads.

## Run Locally

```bash
npm install
npm run dev
```

Open the local URL printed by Next.js. If port 3000 is busy, Next.js will use the next available port.

## Scripts

```bash
npm run lint
npm run build
```

The main page lives in `src/app/page.tsx`.
